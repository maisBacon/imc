## Instalar dependencias

- `yarn` ou - `npm i`

## instalar os plugins

- `eslint e prettier`

## Rodando o projeto

- `yarn start`: inicia o backend.

## Rodando o testes

- `yarn test`

## Rodando DEBUG

Rodar desenvolvimento com `yarn dev` depois (ctrl + shift + d) e selecionar o arquivo de configuração 'Debug' em seguida clickar em Run
